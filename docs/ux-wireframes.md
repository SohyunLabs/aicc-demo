# 뤼튼 AICC SaaS — UX 와이어프레임

## 제품 개요

> **"틀리면 안 되는 상담에서, 검증된 AI 응답을 제공한다"**

Hybrid RAG 기반 출처 추적 가능한 응답 + 산업별 컴플라이언스 엔진 + 성과 기반 과금으로, 금융/보험/이커머스에서 법적/재무적 리스크 없는 AI 상담을 실현하는 AICC SaaS 제품이다.


## 와이어프레임 읽기 가이드

본 문서는 뤼튼 AICC SaaS의 3대 핵심 화면을 고밀도 ASCII 와이어프레임으로 설계한다.

| 화면 | 대상 사용자 | 목적 |
|------|-----------|------|
| **UX-01: 보이스봇 관리 콘솔** | 고객사 운영 관리자 (관리 콘솔 PM, CS 팀장) | 시나리오 구성, 성과 모니터링, AI 지식 베이스 관리 |
| **UX-02: Agent Assist 실시간 보조** | 상담원 | 통화 중 실시간 전사/추천 응답/컴플라이언스 경고 |
| **UX-03: 셀프 온보딩** | 신규 고객사 담당자 | 프로필 등록 -> 산업 템플릿 -> 테스트 콜 -> 결제까지 셀프 서비스 |


## 주석 규약 (Wireframe Annotation Convention)

와이어프레임 내 시각적 표기법:

| 표기법 | 의미 | 예시 |
|--------|------|------|
| (1)(2)(3) | 태스크 번호 마커 | (1) 시나리오 노드를 드래그하여 추가 |
| `[버튼 라벨]` | 클릭 가능한 버튼/링크 | `[시나리오 배포]` |
| `[*강조 버튼*]` | Primary CTA (accent 색상) | `[*테스트 콜 시작*]` |
| `[!경고!]` | Destructive/경고 요소 | `[!면책조항 설명 필요!]` |
| `>>>텍스트<<<` | 시선 유도/추천 강조 | `>>>추천<<<` |
| `(설명 텍스트)` | 비활성/보조 텍스트 (muted) | `(아직 데이터 없음)` |
| `+---+` | 카드/패널 경계선 | 박스 레이아웃 |
| `======` | 섹션 구분선 (카드 내) | 패널 내부 영역 구분 |
| `-->` | 플로우 방향/연결 | 노드 간 엣지, 스텝 진행 |
| `[v]` | 완료 상태 체크 | 스텝 위자드 완료 단계 |

## UX-01: 보이스봇 관리 콘솔

UX-01은 고객사 운영 관리자(관리 콘솔 PM, CS 팀장)가 보이스봇 시나리오를 구성하고, 성과를 모니터링하며, AI 지식 베이스를 관리하는 통합 콘솔이다. GNB 5대 영역(대시보드, 시나리오 빌더, 모니터링, 지식 관리, 설정)으로 구성되며, 기존 AdminPanel(AX PM) 대비 시각적/기능적 개선을 핵심 목표로 한다.


### 기존 AdminPanel 대비 개선 요약

| 영역 | 기존 AdminPanel (AX PM) | 신규 관리 콘솔 |
|------|------------------------|--------------|
| 시나리오 관리 | 텍스트 기반 프롬프트/토픽 목록 | 비주얼 노드-엣지 캔버스 플로우 빌더 |
| 성과 모니터링 | 기본 통계 (수동 샘플링 3-5%) | 실시간 KPI 대시보드 (전수 AI 분석) |
| 지식 관리 | 프롬프트 텍스트 직접 편집 | 문서 업로드 + 자동 RAG 인덱싱 + 토픽 관리 |

- **(1) 해결률 확인:** 현재 AI 자동 해결률 65%를 확인하고, 목표 70% 대비 진행 상황을 파악한다

- **(4) 컴플라이언스 현황 점검:** 전수 검사 기반 위반률 0.3%를 확인하고, 전월(0.5%) 대비 개선을 파악한다

| `/ \` | 의도 분류 노드 (마름모) | 고객 발화 기반 의도 분기 | 적금 문의 / 대출 문의 / 카드 문의 / 기타 |

| `< >` | 상담원 전환 노드 (육각형) | 처리 불가 시 인간 상담원 연결 | "복합 문의입니다. 전문 상담원을 연결합니다." |
| `(( ))` | 종료 노드 (이중 원) | 통화 종료 처리 | 통화 종료 |

**엣지(연결선) 조건 라벨:**
- "고객 의도: 적금 문의" --> 적금 응답 노드
- "고객 의도: 대출 문의" --> 상담원 전환 노드 (복합 상품은 상담원 처리)
- "고객 의도: 카드 문의" --> 카드 응답 노드
- "고객 의도: 기타" --> 기타 응답 노드 (기본 안내 후 상담원 전환 옵션)

**핵심 사용자 태스크:**

- **(1) 노드 드래그로 시나리오 흐름 구성:** [+ 노드 추가] 버튼으로 원하는 노드 유형을 선택하고, 캔버스에 드래그하여 시나리오 흐름을 시각적으로 구성한다
- **(2) 노드 클릭으로 응답 내용 편집:** 캔버스의 노드를 클릭하면 우측 속성 패널이 열리며, 인사말/응답 내용/음성 설정 등을 편집한다
- **(3) 엣지로 분기 조건 설정:** 노드 간 연결선(엣지)을 드래그하여 분기 조건을 설정하고, 고객 의도별 응대 경로를 정의한다
- **(4) 배포 버튼으로 즉시 적용:** [*시나리오 배포*] CTA 버튼으로 편집된 시나리오를 즉시 프로덕션에 적용한다

**기존 AdminPanel 대비 개선:**

> **기존:** 텍스트 목록에서 프롬프트/토픽을 나열 --> 전체 흐름 파악 불가. 시나리오가 복잡해지면 어떤 경로로 고객이 이동하는지 관리자가 추적할 수 없었다.
>
> **신규:** 비주얼 노드 캔버스에서 시나리오 전체 흐름을 한눈에 파악하고 드래그로 편집. 의도 분류 -> 응답 -> 전환 -> 종료까지의 모든 경로가 시각적으로 표현되어, 관리자가 시나리오의 완결성과 누락된 경로를 즉시 확인할 수 있다.

- **(2) AHT로 도입 전후 효율 비교:** AI 도입 전 기준선(6분10초) 대비 현재 4분38초로 25% 단축 효과를 확인한다. 주간 추이 차트로 효율 개선의 지속성을 검증한다
- **(3) CSAT로 고객 만족도 트렌드 모니터링:** 고객 만족도 83점(전월 대비 +2점)과 5점 만족도 분포를 확인한다. AI 도입이 고객 경험에 미치는 영향을 정량적으로 추적한다
- **(4) 컴플라이언스로 전수 검사 기반 위반 감지 현황 파악:** 위반률 0.3%(전월 0.5%에서 개선)를 전수 검사 기반으로 확인한다. 기존 수동 샘플링(3-5%)에서 100% 전수 검사로 전환된 효과를 파악한다

## UX-02: Agent Assist 실시간 보조 화면

UX-02는 상담원이 고객 통화 중 실시간으로 AI 보조를 받는 화면이다. 통화 전사, 추천 응답, 고객 정보, 감정 분석, 컴플라이언스 경고를 하나의 화면에서 동시 확인한다. 보이스봇이 처리하지 못한 상담을 상담원에게 전환할 때, 보이스봇 대화 맥락을 자동 인수하여 고객이 같은 말을 반복하지 않는 연속적 경험을 제공한다.

|  | 전환 사유     | 위약금 상세 계산 요청 (복합 문의)           | |
|  +------------------------------------------------------------+ |

|  고객 감정 상태: 중립 (불안 요소 감지)                           |

**맥락 인수 후 상담원 화면 상태:**
- 좌측 패널: 보이스봇 요약 카드 활성 + 실시간 전사 시작
- 중앙 패널: 미완료 항목(위약금 상세) 기반 추천 응답 자동 로드
- 우측 패널: 고객 정보(김OO / VIP / 정기적금 5,000만원) 자동 표시


### UX-02 핵심 사용자 태스크

1. **(1) 실시간 전사에서 고객 발화를 즉시 확인하고 대화 맥락을 파악한다** -- 좌측 패널의 전사 영역에서 고객과 상담원의 발화가 실시간으로 교대 표시된다. 핵심 키워드가 하단 요약 영역에 자동 추출되어, 긴 통화에서도 대화의 핵심을 놓치지 않는다.
2. **(2) AI 추천 응답을 확인하고 [*추천 응답 사용*] 버튼으로 즉시 활용한다** -- 중앙 패널에 RAG 기반 응답이 출처 문서(정기적금 약관 제12조)와 함께 표시된다. 신규 상담원도 베테랑 수준의 정확한 안내가 가능하다.
3. **(3) 컴플라이언스 경고 발생 시 해당 규정을 확인하고 필수 설명을 수행한다** -- 위반 감지 즉시 중앙 패널에 경고 배너가 표시되고, 전사 텍스트 내 위반 발화에 `[!! 설명의무 누락 !!]` 하이라이트가 표시된다. 상담원이 필수 설명을 수행하면 자동 해제된다.
4. **(4) 우측 고객 정보에서 고객 등급, 거래 이력을 참조하여 맞춤 응대한다** -- VIP 고객의 경우 우선 처리, 거래 이력 기반 맞춤 안내(정기적금 5,000만원 보유 고객에게 대체 상품 제안 등)가 가능하다.
5. **(5) 감정 분석 추이를 모니터링하여 고객 불만이 확대되기 전에 대응한다** -- 감정 게이지가 "불안 --> 안심" 전환 중임을 실시간으로 표시하고, 추이 그래프로 통화 시작부터의 감정 변화를 모니터링한다.

> **UX-02 핵심 가치:** 상담원이 탭 전환 없이 하나의 화면에서 통화 전사/추천 응답/컴플라이언스 경고/고객 정보/감정 분석을 동시에 확인하며, 보이스봇에서 전환된 상담의 맥락을 자동으로 인수받아 고객이 같은 말을 반복하지 않는 연속적 경험을 제공한다.

### UX-03 전체 태스크 요약

고객사는 프로필 등록(1분) > 산업 선택(2분) > 문서 업로드(5-10분) > 시나리오 확인(5분) > 테스트 콜(5분) > 결제(2분)로 총 약 20-30분 내 서비스를 시작할 수 있다. 구축형(6-12개월 PoC) 대비 극적으로 단축된 온보딩 경험이 SaaS 전환의 핵심 가치이다.

## 부록: 기존 AdminPanel 대비 개선 요약

뤼튼 AICC SaaS 콘솔이 기존 AdminPanel(AX PM) 대비 어떤 점에서 개선되었는지를 6개 영역으로 요약한다.

| 영역 | 기존 AdminPanel (AX PM) | 신규 SaaS 콘솔 | 개선 효과 |
|------|------------------------|--------------|----------|
| 시나리오 관리 | 텍스트 프롬프트/토픽 목록 | 비주얼 노드 캔버스 | 전체 흐름 한눈에 파악, 비기술자도 편집 가능 |
| 성과 모니터링 | 수동 샘플링 (3-5%) | 실시간 전수 AI 분석 | 100% 커버리지, 즉시 이상 감지 |
| 지식 관리 | 프롬프트 직접 편집 | 문서 업로드 + 자동 RAG 인덱싱 | 비기술자 문서 관리, 출처 추적 자동화 |
| 컴플라이언스 | 사후 QA | 실시간 인라인 경고 | 위반 사전 차단, 규정 조항 즉시 안내 |
| 상담원 보조 | 별도 도구 참조 | 통합 3칸럼 화면 | 탭 전환 없이 모든 정보 동시 확인 |
| 온보딩 | 구축형 6-12개월 | 셀프서비스 20-30분 | 도입 전 AI 품질 직접 검증 |
