   - 고객 감정 상태 (중립/긍정)
4. 상담원 화면에 "고객이 A상품과 B상품을 비교 중이며, 3번째 상품 추가 비교 요청" 요약 표시
5. 상담원은 처음부터 다시 묻지 않고 바로 상담 이어감

시너지 효과 문장: "고객은 같은 말을 반복하지 않고, 상담원은 맥락 파악에 시간을 쓰지 않는다."
